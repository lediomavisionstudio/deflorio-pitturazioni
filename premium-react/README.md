# Deflorio Pitturazioni - Premium React

Base moderna per usare:

- GSAP per animazioni cinematografiche
- Lenis per scroll fluido
- Motion per microanimazioni
- React Three Fiber + Drei per 3D
- componenti stile shadcn/ui, Aceternity UI e Magic UI

## Avvio

```bash
npm install
npm run dev
```

Poi apri l'URL mostrato da Vite.
